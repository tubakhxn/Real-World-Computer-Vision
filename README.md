# Computer Vision Systems

This repository contains three computer vision systems built using Python, OpenCV, YOLO, and real-world video analysis.

The projects focus on industrial monitoring, safety intelligence, and autonomous perception.

Built for educational and research purposes only.

---

## 1. Warehouse Forklift Safety System

### Overview

Computer vision system for monitoring warehouse operations and identifying potential safety risks involving forklifts and workers.

### Features

* Forklift detection
* Worker detection
* Near-miss monitoring
* Safety zone analysis
* Worker tracking
* Forklift tracking
* Risk scoring
* Occupancy analytics
* Activity heatmaps
* Safety dashboard

### Run

```bash
python warehouse_forklift_safety.py
```

---

## 2. Port Container Intelligence Platform

### Overview

AI-powered logistics intelligence platform for analyzing container terminal operations and yard activity.

### Features

* Container detection
* Container tracking
* Crane monitoring
* Truck monitoring
* Yard occupancy analysis
* Congestion detection
* Throughput analytics
* Density heatmaps
* Activity monitoring
* Operations dashboard

### Run

```bash
python port_container_intelligence.py
```

---

## 3. Self-Navigating Vision Driving System

### Overview

Vision-only autonomous driving perception system that analyzes road scenes using camera input.

### Features

* Vehicle detection
* Pedestrian detection
* Lane detection
* Depth estimation
* Driving risk analysis
* Collision awareness
* Traffic monitoring
* Path visualization
* Bird's-eye-view analytics
* Driving dashboard

### Run

```bash
python self_navigating_vision_driving.py
```
